Production-ready fullstack scaffold for Abyei Students in Rwanda community site (Musanze).

Tech stack
- Frontend: Next.js (App Router) + TypeScript
- Styling: Tailwind CSS
- Backend: Node.js + Express (separate service)
- DB: PostgreSQL (Railway)
- ORM: Prisma
- Auth: JWT role-based (Admin, Editor, Member)
- Forms: React Hook Form + Zod

Deliverables included in this scaffold
- Folder structure
- Prisma database schema (prisma/schema.prisma)
- Backend: auth, members, news, events routes with Zod validation
- Frontend: Next.js App Router pages and a Hero section with admin login
- Example `.env` file and deployment notes

See [DEPLOYMENT](./DEPLOYMENT.md) for Vercel / Railway instructions.
